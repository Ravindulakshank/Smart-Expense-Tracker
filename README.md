# Desktop Smart Expense Tracker App 💰📊
A desktop application built with Python and Tkinter that helps users track their daily expenses with a clean Graphical User Interface (GUI).

## Features & Concepts Covered
- **Graphical User Interface (GUI)**: Built using Python's built-in `tkinter` library (Buttons, Entry fields, Labels, and Messageboxes).
- **File Handling**: Dynamically reads and writes expense data into a local `expenses.txt` file, ensuring data persists even after closing the app.
- **Error & Exception Handling**: Uses `try-except` blocks (`ValueError`) to catch invalid inputs (e.g., typing letters instead of numbers for the amount) and prevents the app from crashing.

## How to Run
